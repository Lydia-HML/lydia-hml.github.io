動態網頁程式設計Web Dynamic Programming

本課程旨在培養學生了解HTML、CSS、JavaScript組合的網頁運作原理，課程以熟悉JavaScript與JQuery基本語法為主要目的，並能靈活運用現有插件增加網頁的動態效果，評量方式則以實作演練為主。 This course aims to teach students to understand how it works page HTML, CSS, and JavaScript combination. The primary purpose, of course, is to become familiar with the programming of JavaScript and JQuery. This course will also utilize the existing plug-in adds dynamic effects page. Finally, the assessment exercise is based on an implementation approach.
